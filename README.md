# Ringside: Last Bell

Browser boxing game with an open-world story mode, AI fights, local 2-player mode, and PeerJS lobby support.

## Play locally

Open `index.html` in a browser, or serve the folder with a simple local server.

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.

## Files

- `index.html` - game page and UI
- `styles.css` - layout and visual styling
- `game.js` - game logic
- `assets/` - portraits and environment art

## Controls

- Move: WASD / arrow keys / touch pad
- Guard: Space or L
- Light punch: J
- Power punch: K
- Body punch: U
- Slip: Shift
